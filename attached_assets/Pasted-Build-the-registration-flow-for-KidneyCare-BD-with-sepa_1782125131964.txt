Build the registration flow for “KidneyCare BD” with separate pages for **Patient**, **Doctor**, and **Community Health Worker (CHW)**. Keep the design minimal, functional, and fully aligned with the project's mobile‑first, bilingual PWA style.

- All pages must be responsive (mobile‑first), with max‑width 480px centered on larger screens.
- Use the project color palette: primary #1A6B8A, error #E74C3C, success #2ECC71, white cards with subtle shadow.
- All labels, placeholders, validation messages, and buttons must be **bilingual** (English / বাংলা) and switch instantly via the language toggle (use i18n).
- Touch targets ≥48px, proper mobile input keyboards (inputmode).
- Forms must work inside the PWA (service worker caches the registration page shell; actual submission requires online).
- After successful registration, redirect to login page with a success message.

COMMON FIELDS (ALL ROLES)

- Full Name (in English, optional Bengali)
- Email (unique, used for login)
- Phone Number (Bangladesh format, 01XXXXXXXXX)

ROLE‑SPECIFIC FIELDS

### Patient Registration
- Age (numeric, 1‑120)
- Sex (radio: Male / Female)
- Weight in kg (numeric, 30‑300)
- Do you have diabetes? (Yes/No/Don't Know)
- Do you have hypertension? (Yes/No/Don't Know)
- Family history of kidney disease? (Yes/No/Don't Know)
- Which area do you live in? (Urban / Rural) — directly feeds risk scoring

### Doctor Registration
- Medical Registration Number (BMDC number, text field)
- Specialty (dropdown: Nephrologist, General Physician, Diabetologist, Other)
- Hospital/Clinic Name (text)
- Years of Experience (numeric)

### CHW Registration
- National ID or CHW ID (text)
- Organization/NGO Name (text, optional)
- Assigned Working Area (free text, e.g., "Rangpur Sadar Union")
